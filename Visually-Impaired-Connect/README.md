# VisionConnect - Visually Impaired Mobility Platform

A React-based web application connecting visually impaired individuals with volunteers for safe outdoor travel assistance.

## Features

- **Accessible Design**: Voice-enabled navigation, biometric authentication
- **Multi-language Support**: English, Hindi, Tamil, Telugu, Bengali
- **Dual User Types**: Visually impaired users and volunteers
- **Secure Authentication**: Face ID and fingerprint login
- **Trust & Safety**: Verified volunteer profiles

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm start
```

3. Open http://localhost:3000

## Pages

- **Home**: Landing page with features and use cases
- **Role Selection**: Choose between user or volunteer
- **Sign Up**: Separate flows for users and volunteers
- **Login**: Biometric and phone-based authentication

## Tech Stack

- React 18
- React Router 6
- Tailwind CSS
- Lucide React Icons
